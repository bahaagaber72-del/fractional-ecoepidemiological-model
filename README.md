# Fractional-Order Ecoepidemiological Predator–Prey Model

This repository contains the MATLAB source codes used for the numerical simulations, bifurcation analysis, and optimal control computations presented in the manuscript:

“Fractional-Order Analysis of a Fear-Induced Ecoepidemiological Predator–Prey Model with Optimal Control and Bifurcation Dynamics”.
